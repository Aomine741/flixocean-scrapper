✅ GitHub Actions Auto-Scraper Setup (FlixOcean)

1. Create a GitHub repo and upload all files.
2. Go to "Settings" → "Secrets and variables" → "Actions"
3. Add these 3 secrets:
   - GH_TOKEN = your GitHub token (with repo access)
   - GITHUB_REPO = yourusername/yourrepo (e.g. Aomine741/FlixOcean)
   - GPLINK_API = your GPLinks API key
4. GitHub will run the scraper every 6 hours automatically.
