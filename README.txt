1. Upload this folder to a new GitHub repo (e.g. flixocean-scraper)
2. Go to https://render.com > New > Background Worker
3. Connect your GitHub repo
4. Set Environment Variables:
   - GH_TOKEN = your GitHub personal access token (with repo permission)
   - GITHUB_REPO = yourusername/yourrepo (e.g. Aomine741/FlixOcean)
   - GPLINK_API = your GPLinks API key
5. Deploy and Render will run it automatically
